# pagoflash-prestashop
-- -------------------------------------------------------------------------------------------------
-- Aspectos técnicos
-- -------------------------------------------------------------------------------------------------

Requerimientos
--------------
- PHP 5.4 o superior
- Prestashop 1.5 o superior

Instalación
------------
1. Descarga el plugin
2. Ingresa a tu cuenta admin de prestashop y entra a "Módulos y Servicios"
4. Click a añadir nuevo módulo
5. Carga el plugin de pagoflash descargado en formato .zip
6. Click en instalar

-- -------------------------------------------------------------------------------------------------
-- Configura el plugin
-- -------------------------------------------------------------------------------------------------

El área de configuración del plugin se muestra una vez que le das click a instalar y posee las siguientes opciones configurables:

  - Clave de la aplicación: (Key Token) Ficha única que genera PagoFlash al momento de registrar un punto de venta virtual.

  - Clave Secreta: (Key Secret) Ficha única complementaria que genera PagoFlash al momento de registrar un punto de venta virtual.

  - Callback URL: Solo lectura. El contenido de este campo debe ser copiado y pegado en el campo "URL callback" del formulario de registro del punto de venta virtual en PagoFlash.
  

**¡Listo! Ya tus clientes podrán pagar sus productos a través de PagoFlash.**
